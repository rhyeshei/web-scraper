# Web Scraper Project (Books)

## プロジェクト概要
本プロジェクトは、紀伊國屋書店のWebサイトから書籍のタイトルおよび価格情報を取得し、CSVファイルとして保存するPython製のWebスクレイパーです。

スクレイピング処理を通じて、Webページの構造理解からデータ抽出・加工・保存までの一連の流れを学習することを目的としています。

---

## 作成背景
本プロジェクトは、授業内のPython演習において、講師が作成したテキスト教材を参考にしながら作成しました。  
教材で示されたコード構造や主要ライブラリの使用方法をもとに、実際のWebサイトを対象としたスクレイピング処理を実装しています。

---

## 学習目的
- HTTPリクエストによるWebページ取得の理解
- HTML構造を解析し、必要な情報を抽出する基礎力の習得
- pandasを用いたデータ整形およびCSVファイル出力の実装
- スクレイピング実行時の注意点（規約・負荷）への理解

---

## 使用技術
- **言語**: Python 3.9.6
- **主なライブラリ**:
  - `requests`：WebページのHTML取得
  - `BeautifulSoup`（bs4）：HTMLの解析および要素抽出
  - `pandas`：データの整理およびCSV出力

---

## プログラムの機能
1. 紀伊國屋書店の書籍一覧ページへHTTPリクエストを送信します。
2. HTML内の特定クラス（`.list_area_wrap`）から、書名（画像の `alt` 属性）および価格（`.sale_price`）を抽出します。
3. 抽出したデータを「書名」「価格」の2列を持つデータフレームに変換します。
4. 取得結果を `books.csv` として、UTF-8（BOM付き）形式で保存します。

---

## 実行方法
1. 必要なライブラリをインストールします。
   ```bash
   pip install requests beautifulsoup4 pandas
2. `jp.ipynb` を Jupyter Notebook で開き、上から順にセルを実行します。
3. 実行後、同一ディレクトリ内に `books.csv` が生成されます。
